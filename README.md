# EDA_Guillem_PV
 Se trata de un repositorio creado para la EDA como entrega individual del bootcamp en datascience
# LA FELICIDAD EN EL MUNDO

A partir de la base de datos de la World Happiness Report se ha hecho un análisis exploratorio de datos (EDA) para entender a la perfección la base de datos y lo que nos ofrece en cuanto a información. 


## Contenido
EDA.ipynb: notebook con el análisis presentado.
data: carpeta con los dataset procesados y no procesados.
img: carpeta con figuras e imágenes utilizadas o extraídas del proyecto.
notebooks: notebooks con el desarrollo del proyecto. A continuación se presenta el contenido de estos.
EDA_presentacion.pdf: archivo presentación para la exposición del proyecto.
memoria.pdf: archivo memoria explicando el proyecto.
